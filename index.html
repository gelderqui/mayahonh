Crud PHP y Vue

Este proyecto es un crud (crear,listar,actualizar y eliminar) sobre una tabla aleatoria realizado en PHP y VueJS. PHP se utiliza para hacer la conexion a la base de datos y tambien se utiliza para hacer la logica del crud, VueJs se usa para que se pueda manipular la data en el frontend, para poder conectar tanto el front como el backend se utiliza axios, para la vista de la pagina se utiliza Bootstrap, también se utiliza SweetAlert y Font Awesome todas estas librerias se cargan como cdn.
